Esta es una configuración sencilla y sin plugins ni nada raro para *Neovim*. La he comentado para que se pueda entender si no se es familiar con Neovim. Se puede instalar con un solo comando tal que:

#+begin_src bash
mkdir -p ~/.config/nvim
curl https://github.com/Dorovich/dotfiles/tree/main/.config/nvim.vanilla/init.lua > ~/.config/nvim/init.lua
#+end_src

Si buscas una configuración algo más sofisticada, la mía está [[aquí][https://github.com/Dorovich/dotfiles/tree/main/.config/nvim]].
